Customize your heart icon

You can customize your heart avatar with your own photo using the project files. If you don't have a copy of Photoshop, you can use GIMP or download a 30-day trial from adobe.com Alternatively, you can also use the avatar generator.

Photoshop (.psd)

  1. Open the photo you want to use for your avatar in Photoshop
  2. From the Select menu select All
  3. From the Edit menu select Copy
  4. Open the file HEART_TEMPLATE.psd in Photoshop
  5. Select the layer YOUR IMAGE in the Layers window
  6. From the Edit menu select Paste
  7. Press the v key to activate the Move tool
  8. Click and drag your photo to position it the way you want it
  9. From the Edit menu select Free Transform
  10. Click and drag the one of the image corners while holding down Shift to resize the photo
  11. Click on the small arrow in the layer CHOOSE YOUR COLOR to see the different colors available
  12. Check the checkbox of the layer that contains the color you like to activate the color
  13. From the File menu select Save for Web
  14. Save the file as PNG-24 without transparency

GIMP (.xcf)

  1. Open the file HEART_TEMPLATE.xcf with GIMP
  2. Press the Delete key to remove the existing overlayed image
  3. From the File menu select Open as Layers... and open your own avatar
  4. You can resize the image with the scale tool (Shift+T holding Ctrl)
  5. And the position with the move tool (M)
  6. Ensure your avatar is in "Overlay" mode which is controlled by a dropdown above the layers (Other modes work too, so experiment)
  7. Click on the small plus in the layer CHOOSE YOUR COLOR to see the different colors available
  8. Check the leftmost "eye" checkbox of the layer that contains the color you like to activate the color
  9. When happy export your new avatar: File > Export As... and export your image as a PNG
